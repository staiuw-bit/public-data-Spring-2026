# Samantha Tai's D3 HW2
Below you will see a line graph with two lines that shows the Seattle Tacoma Airport station min and max temperature observations. This data is provided the NOAA (National Center For Environmental Information). 


I filtered the data in May 2025. Since the dataset was not available in CSV format, I manually copied the date, minimum temperature, and maximum temperature into an Excel sheet and then saved it as a CSV file. Because the dataset contains 31 days of observations, manually entering all the data would be too time-consuming, so I only entered the first ten days. For reference, a screenshot of the original data table is also included in this folder.
