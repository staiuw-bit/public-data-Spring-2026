# Samantha Tai's First Flourish HW 

Below you will find a bar chart created by me using *Seattle Open Data* platform. 

This graph is generated using the Community Reporting Areas (2020) - Redistricting Data 1990-2020 Dataset. 

I cleaned the dataset using R studio and filtered the columns to only compare Districts and 1990 White not Hispanic or Latino Population variables. 

Takeaways from this visual: 
1. In 1990, White population group was concentrated in specific residential districts, especially in the north and northwest parts of the city. 
2. Urban areas like Downtown had much smaller residential populations.
